API EndPoints.

Metodo GET

El método GET devuelve la informacion de la/s tarea/s.

  • /tasks
  • /tasks/:id

Metodo POST

El método POST recibe los datos necesarios para agregar una tarea.

  • /tasks

Metodo PUT

El método PUT recibe los datos que desea actualizar de una tarea en especifico.

  • /tasks/:id

Metodo DELETE

El método DELETE elimina una tarea por su id

  • /tasks/:id
